------------------------------------------------------------
Trigger & Response
------------------------------------------------------------

Webhook (n8n-nodes-base.webhook)  
Method: POST   
Use: Receives the incoming resume/job data.

Respond to Webhook (n8n-nodes-base.respondToWebhook)  
Use: Sends the final, formatted analysis back to the caller.

------------------------------------------------------------
Google Drive
------------------------------------------------------------

Upload Resume (n8n-nodes-base.googleDrive, operation: upload)  
Use: Stores the uploaded resume PDF in your Drive folder.

Download (n8n-nodes-base.googleDrive, operation: download)  
Use: Pulls the just-uploaded resume back for processing.

Download Role (n8n-nodes-base.googleDrive, operation: download)  
Use: Fetches the job description text file from Drive.

------------------------------------------------------------
File/Text Extraction
------------------------------------------------------------

Extract Data (n8n-nodes-base.extractFromFile, operation: pdf)  
Use: Converts the PDF resume to plain text.

Extract from File (n8n-nodes-base.extractFromFile, operation: text)  
Use: Reads the job description text file.

------------------------------------------------------------
AI / LangChain / OpenAI
------------------------------------------------------------

Mind (@n8n/n8n-nodes-langchain.lmChatOpenAi)  
Use: The LLM connector used by the agent.

Resume Analysis (@n8n/n8n-nodes-langchain.agent)  
Prompt type: define  
Use: Compares the resume against the job description and produces the structured evaluation.

Parse Output (@n8n/n8n-nodes-langchain.outputParserStructured)  
Use: Forces the agent’s answer into a strict JSON schema (strengths, weaknesses, risk/reward, score, justification).

Information Extractor (@n8n/n8n-nodes-langchain.informationExtractor)  
Use: Pulls first name, last name, and email from the resume text.

Formatting Output (@n8n/n8n-nodes-langchain.lmChatOpenAi)  
Use: (Optional) Further prettifies/normalizes the final text. Could be removed if you don’t need it.

------------------------------------------------------------
Data Shaping
------------------------------------------------------------

Prompt (n8n-nodes-base.set)  
Use: Builds the exact prompt string that the agent consumes.

Edit Fields (n8n-nodes-base.set)  
Use: Maps/assembles the final fields (date, strengths, weaknesses, scores, justification) for the webhook response.

------------------------------------------------------------
HTTP Request Node?
------------------------------------------------------------

None used. All integrations (Google Drive, OpenAI) are handled with built-in nodes. No external HTTP Request node is present.
